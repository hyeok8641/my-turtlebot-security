## 👥 3조 조원 라인업 및 역할 분담
* **차재혁 (팀장)** : ROS 2 `patrol_node` 상태머신 설계, 고속 비상 브레이크 인터럽트 알고리즘 구현, 저장소 형상 관리
* **이종한** : E2E 통합 인터페이스 연동 총괄, 가이드라인 규격 기반 단위/통합 테스트 케이스 수행
* **이준제** : 터틀봇3 Burger 하드웨어 Bringup, `tmux` 기반 원터치 자동화 기동 인프라 스크립트 구축
* **김민상** : 스마트폰 독립 카메라 연동, 웹 브라우저 기반 Edge AI 화재 감지 알고리즘 및 연속 프레임 디바운스 필터 구현
* **정현지** : `roslibjs` 기반 웹 관제 대시보드 UI 구현, 비상 사이렌 오디오 시스템 및 위험 상황 웹 반전 연동

### 1. 정밀 지도 데이터 다운로드 및 배치 (필수)
본 프로젝트는 실습실의 정밀 지도 데이터를 홈 디렉토리(`~/`) 기준으로 참조합니다.
* 저장소 내의 **`my_50_room_map.yaml`** 파일과 **`my_50_room_map.pgm`** 파일 2개를 다운로드합니다.
* 다운로드한 파일들을 우분투 **홈 폴더 경로(`~/`) 바로 아래**로 복사(이동)해 주십시오.
